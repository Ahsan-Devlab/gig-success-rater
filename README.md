# 💼 Freelance Gig Success Rater

A machine learning web application that predicts if a freelance contract will be completed on time or run late using a Random Forest Classifier.

🔗 **Live App Demo:** [Freelance Gig Success Rater](https://ml-project-06-gig-success-rater.streamlit.app/)

## 🧠 Concepts Covered
* **Random Forest:** Training an ensemble learning method that builds multiple decision trees and merges them together for a more accurate and stable prediction.
* **Feature Importance:** Extracting and visualizing the exact weight the AI assigns to different variables (Budget vs. Deadline vs. Client History).
* **Binary Classification:** Predicting a strict "Yes/No" (On Time vs. Late) outcome based on historical gig data.

## 🛠️ Tech Stack
* **Language:** Python
* **ML Libraries:** Scikit-Learn
* **Data Processing:** Pandas
* **Visualization:** Matplotlib
* **Deployment & UI:** Streamlit
